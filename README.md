**How to Run C Sharp files in local PC**

##Clone the repository

## Open the solution:

- Double-click the `.sln` file
- Or open the `.csproj` in Visual Studio

## Build the project:

- Use `Ctrl + Shift + B`
- Or select `Build > Build Solution` from the menu

## Run the app:

- Press `Ctrl + F5`
- Or click `Start Without Debugging`
